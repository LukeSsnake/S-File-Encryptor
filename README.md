# Android File Encryption – Code Snippet

Trecho de código extraído do app **S File Encryptor** (Android).

Este repositório contém apenas um exemplo educacional de criptografia de arquivos, focado em:

- AES
- PBKDF2
- HMAC
- AsyncTask
- Android Java

## ⚠️ Aviso
Este código **não é o app completo da Play Store**.
Não contém chaves reais nem configurações sensíveis.

## Objetivo
Demonstrar a lógica principal de criptografia para estudo e referência.
